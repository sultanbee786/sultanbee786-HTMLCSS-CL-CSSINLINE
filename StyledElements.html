CSS styling

This exercise relies on inline CSS - style rules applied to an individual element, within the element itself. In total, there are three ways to introduce CSS: inline (as shown here), embedded (written in the HTML head), and external (written in a separate file.) External CSS is generally preferred because separating style rules into its own file results in a more efficient separation of concerns.
